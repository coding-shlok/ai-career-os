# AI Career OS

## Vision

Build an AI-powered career operating system that helps students and professionals manage resumes, skills, job applications, interview preparation, and career growth in one unified platform.

---

## Problem Statement

Students often use multiple tools for resumes, job tracking, skill development, and interview preparation. There is no centralized, intelligent system that understands a user's career profile and provides personalized recommendations.

AI Career OS aims to become a single platform that assists users throughout their career journey.

---

## Features

* User authentication and profile management
* Resume upload and parsing
* Skill extraction using AI
* Career recommendations
* Job tracking dashboard
* Interview preparation assistant
* Personalized learning roadmap
* Semantic search using vector databases
